# 让我喵喵 会员解锁脚本 - 使用指南

## 📱 支持平台
- ✅ Loon (推荐)
- ✅ Quantumult X (QX)
- ✅ Surge
- ⚠️ Egern (部分支持)

---

## 🚀 方法一：Loon 傻瓜式安装（推荐）

### 步骤 1：订阅脚本
1. 打开 Loon
2. 点击底部菜单 → **脚本**
3. 进入 **订阅脚本** 页面
4. 点击 **+** 添加订阅
5. 粘贴以下链接：
```
https://raw.githubusercontent.com/Yu9191/Rewrite/main/rwmm_fixed.js
```
6. 命名如「让我喵喵修复版」
7. 保存并启用

### 步骤 2：配置 Mitm
1. 进入 **Mitm** 页面
2. 启用 Mitm
3. 安装并信任 CA 证书
4. 在 hostname 中添加：
```
www.pdreamer.com
```

### 步骤 3：验证
1. 打开「让我喵喵」App
2. 查看会员状态是否显示 **SVIP** 或 **已开通**

## 🚀 方法二：手动导入（无需订阅）

### 步骤 1：复制脚本内容
复制下方 **修复版脚本** 的全部内容：

```javascript
/*
#!name=让我喵喵 - 会员解锁 (修复版)
*/

const path = $request.url;
const body = getFakeVipResponse();
$done({ body: body });

function getFakeVipResponse() {
    const now = new Date();
    const future = new Date(now.getFullYear() + 10, now.getMonth(), now.getDate());
    const expireTime = future.toISOString().replace('T', ' ').substring(0, 19);
    const currentTime = now.toISOString().replace('T', ' ').substring(0, 19);
    
    return JSON.stringify({
        code: 200,
        msg: "success",
        data: {
            userId: 1145141919,
            nickname: "SVIP会员",
            vip: true,
            vipLevel: "SVIP",
            isVip: true,
            expireTime: expireTime,
            startTime: currentTime,
            vipDay: 9999,
            totalVipDay: 9999,
            member: {
                status: 1,
                level: 3,
                expire: expireTime
            }
        }
    });
}
```

### 步骤 2：Loon 配置
1. 打开 Loon → **脚本**
2. 点击 **Rewrite Local** 或 **HTTP Response**
3. 点击 **+** 新建
4. 粘贴脚本
5. 保存

### 步骤 3：添加 MITM
1. 设置 → **MITM**
2. 启用
3. **hostname** 添加：`www.pdreamer.com`

## ⚠️ 常见问题

### Q1: 脚本没生效？
- 检查 Mitm 是否启用
- 检查 hostname 是否正确添加
- 重启 App 或重新打开 App
- 查看日志排查

### Q2: 显示"网络错误"？
- 确保证书已安装并信任
- 检查 VPN/代理是否正常工作

### Q3: 如何确认脚本生效？
- 查看 Loon 日志
- 搜索 `pdreamer` 关键词
- 应该看到类似：`[让我喵喵] 拦截响应: https://www.pdreamer.com/...`
